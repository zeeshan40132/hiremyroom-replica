HIRE MY ROOM - RENTAL PROPERTY WEBSITE
=====================================

PROJECT OVERVIEW
----------------
Hire My Room is a comprehensive rental property website that allows users to browse, search, and book rooms and apartments. The platform features a modern, responsive design with multiple language support and interactive features.

TECHNOLOGY STACK
----------------
- Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
- Backend: Node.js, Express.js
- Fonts: Google Fonts (Poppins, Georgia)
- Icons: Font Awesome 6.4.0
- Video Support: HTML5 Video Player

INSTALLATION & SETUP
--------------------
1. Prerequisites:
   - Node.js (v14 or higher)
   - npm (Node Package Manager)

2. Installation Steps:
   a. Clone or download the project files
   b. Open terminal/command prompt in project directory
   c. Install dependencies: npm install
   d. Start the server: node server.js
   e. Open browser and navigate to: http://localhost:3000

3. Dependencies:
   - express: Web framework
   - path: File path utilities
